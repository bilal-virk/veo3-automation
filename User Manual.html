üì¶ Extension Installation

Step 1: Extract the Extension

  1. Download the extension ZIP file
  2. Right-click the ZIP file and select "Extract All" or "Extract Here"
  3. Remember the folder location where you extracted the files

Step 2: Install in Chrome

  1. Open Google Chrome browser
  2. Type chrome://extensions in the address bar and press Enter
  3. Turn ON "Developer mode" (toggle switch in top-right corner)
  4. Click the "Load unpacked" button
  5. Browse to the folder where you extracted the files
  6. Select the folder and click "Select Folder"
  7. The extension is now installed! You'll see it in your extensions list

Step 3: Pin the Extension (Optional)

  1. Click the puzzle icon (üß©) in Chrome toolbar
  2. Find "Flow Automation veo3" in the list
  3. Click the pin icon (üìå) next to it
  4. Now you can access it easily from your toolbar

üîß n8n Workflow Setup

n8n is the automation platform that processes your video requests and generates AI prompts. Follow these steps to set it up:

Step 1: Import the Workflow

  1. Log into your n8n account at https://app.n8n.cloud (or your self-hosted instance)
  2. Click "Workflows" in the left sidebar
  3. Click the "+" button or "Add Workflow"
  4. Click the three dots (‚ãÆ) in the top-right corner
  5. Select "Import from File"
  6. Choose the Veo3_Master.json file provided to you
  7. The workflow will be imported with all nodes

Step 2: Configure Credentials

You need to set up two credentials:

A. Google Service Account
  1. Click on the "+" node
  2. Click on the "Credential" dropdown
  3. Select "Google Service Account API"
  4. Enter your service account JSON credentials
  5. Save with name: Google Service Account API
B. OpenAI API
  1. Click on the "+" node
  2. Click on the "Credential" dropdown
  3. Select "OpenAI"
  4. Enter your OpenAI API key
  5. Save with name: OpenAi account
üí° Getting Credentials

Google Service Account: Create one in Google Cloud Console ‚Üí IAM & Admin ‚Üí Service Accounts or watch this video

OpenAI API Key: Get from https://platform.openai.com/api-keys

Step 3: Start the Workflow

  1. Make sure all nodes are properly connected (they should be after import)
  2. Click "Save" in the top-right corner
  3. Click the "Activate" toggle switch to turn it ON
  4. The workflow is now active and ready to process requests!

Step 4: Execute Workflow

  1. Click on the "Execute workflow"
  2. Fill in the form with:
    • Google Sheets URL: Your spreadsheet URL
    • Sheet name: Sheet1 (where you add video ideas)
    • Recheck Time: 60 seconds (recommended)
  3. Submit & CLose the form - this starts the automation loop!
‚úÖ What happens next?

The workflow will now check your Sheet1 every 60 seconds for new rows, process them with OpenAI, and add generated prompts to the n8n sheet.

Understanding the Workflow

The workflow consists of these key components:

Node Purpose
On form submission Trigger - starts the workflow with your sheet details
Read Client Sheet Reads rows from your Sheet1
If Filters only rows that aren't marked "Done"
Prepare OpenAI Prompts Creates AI prompts from your video descriptions
Generate with OpenAI Calls OpenAI GPT-4 to generate detailed video prompts
Parse AI Response Extracts segments from OpenAI response
Write to n8n Sheet Saves generated prompts to n8n sheet
Mark Row as Done Updates Sheet1 status to "Done"
Wait Waits 60 seconds before checking again

How it works:
  1. You add rows to Sheet1

  4. n8n adds processed data to n8n sheet
  5. Extension reads from n8n sheet and creates videos
  6. Extension marks rows as "Done" when complete

üé¨ How to Use the Extension

Step 1: Add Video Ideas to Sheet1

  1. Open your Google Sheet
  2. Go to Sheet1 tab
  3. Add rows with your video ideas (fill all required columns)
  4. Leave the Status column empty
  5. Save the sheet

Step 2: Wait for n8n to Process

  • n8n checks Sheet1 every 60 seconds
  • It processes unprocessed rows (where Status is empty)
  • Generates AI prompts using OpenAI
  • Adds them to the n8n sheet
  • Marks Sheet1 rows as "Done"
üí° Check Progress

You can check the n8n sheet tab to see processed prompts appearing.

Step 3: Open VEO3

  1. Go to: https://labs.google/fx/vi/tools/flow
  2. Make sure you're logged into your Google account
  3. Keep this tab open (don't close it)

Step 4: Start Extension Automation

  1. Click the extension icon in Chrome toolbar
  2. You'll see the Automation tab with stats:
    • Total Rows: All rows in your n8n sheet
    • Processed: Rows already completed
  3. Click the "Start Automation" button

Step 5: Watch It Work

The extension will now:

  • Read each unprocessed row from the n8n sheet
  • Create the video on VEO3 using the AI-generated prompt
  • Download videos automatically
  • Mark the row as "Done" in n8n sheet
  • Move to the next row
  • Continue checking every 60 seconds for new rows
üí° Real-time Progress

Watch the progress bar and activity log in the extension popup to see what's happening!

Step 6: Stop Anytime (Optional)

  • Click "Stop Automation" button to pause
  • You can resume later - it remembers where it stopped
  • Already processed rows are marked "Done" and won't be repeated

‚ùì Troubleshooting

Extension Issues

Problem: "VEO3 Tab Required" Warning

Solution: Open https://labs.google/fx/vi/tools/flow in a new tab and keep it open

Problem: "Cannot access sheet" Error

Solutions:

  • Make sure sheet is shared with: google-sheet-reader@civil-tempo-479514-t9.iam.gserviceaccount.com
  • Check the sheet URL is correct
  • Verify the n8n sheet exists (run n8n workflow first)
Problem: Videos Not Downloading

Solutions:

  • Check VEO3 site has "Allow" permission for automatic downloads
  • Check Chrome download settings aren't blocking downloads
  • Verify you have enough disk space
  • Check your antivirus isn't blocking downloads
Problem: No Rows Being Processed

Solutions:

  • Check that n8n has processed your Sheet1 rows
  • Verify rows in n8n sheet don't already say "Done"
  • Make sure the extension is connected to the right sheet
  • Test the connection in Settings tab

n8n Workflow Issues

Problem: Workflow Not Processing Sheet1

Solutions:

  • Make sure workflow is Activated (toggle ON)
  • Check you submitted the initial form with sheet details
  • Verify Google Service Account credential is set up correctly
  • Make sure Sheet1 exists and is shared with service account
Problem: OpenAI Errors

Solutions:

  • Check your OpenAI API key is valid and has credits
  • Verify the OpenAI credential is set up correctly in n8n
  • Check OpenAI API usage limits
Problem: n8n Sheet Not Created

Solutions:

  • The workflow creates it automatically on first run
  • Check the workflow execution logs for errors
  • Verify Google Sheets credential has Editor permissions
  • Make sure the sheet is shared with the service account

Google Sheets Issues

Problem: "Permission Denied" Errors

Solution: The service account email must have Editor access, not just Viewer

Problem: Data Not Updating

Solutions:

  • Wait for the check interval (60 seconds default)
  • Refresh your Google Sheet to see latest changes
  • Check both Sheet1 and n8n sheet tabs
