üîê Step 1: Google Service Account Setup

What is a Service Account?

A Service Account is a special Google account that allows automated systems (like n8n) to access Google Sheets without requiring a human to log in each time.

Creating the Service Account (If Not Already Created)

‚ö†Ô∏è Note

If you already have the service account, you can skip this section and go to "Adding Service Account to n8n".

  1. Go to Google Cloud Console
  2. Select your project or create a new one
  3. Navigate to: IAM & Admin ‚Üí Service Accounts
  4. Click "Create Service Account"
  5. Enter a name (e.g., "n8n-automation")
  6. Click "Create and Continue"
  7. Skip role assignment, click "Continue"
  8. Click "Done"
  9. Click on the created service account
  10. Go to the "Keys" tab
  11. Click "Add Key" ‚Üí "Create new key"
  12. Select "JSON" format
  13. Click "Create" (the JSON file will download)
  14. Save this JSON file securely - you'll need it for n8n

Enable Google Sheets API

  1. In Google Cloud Console, go to "APIs & Services" ‚Üí "Library"
  2. Search for "Google Sheets API"
  3. Click on it and click "Enable"

‚öôÔ∏è Step 2: n8n Workflow Setup

2.1: Add Service Account Credentials to n8n

  1. Log in to your n8n instance
  2. Click on "Credentials" in the left sidebar
  3. Click "Add Credential"
  4. Search for and select "Google Service Account API"
  5. Open the JSON file you downloaded earlier
  6. Copy the Service Account Email and Private Key from JSON file
  7. Paste it in the "Service Account Email" and "Private Key" fields in n8n
  8. Click "Save"
Connection Tested Successfully

You should see your credential in the credentials list.

2.2: Import the n8n Workflow

  1. In n8n, click "Workflows" in the left sidebar
  2. Click "Add Workflow" ‚Üí "Import from File"
  3. Select the provided workflow JSON file: veo3-master-workflow.json
  4. The workflow will be imported and opened
üí° Tip

The workflow file should be provided with this documentation package.

2.3: Configure Workflow Credentials

After importing, you need to link your Google Service Account credential to all Google Sheets nodes:

  1. Click on the "Read Master Sheet" node
  2. Update "Document" By URL input with the URL of your Master Sheet
  3. In the "Credential to connect with" dropdown, select your Google Service Account credential
  4. Repeat this for ALL Google Sheets nodes in the workflow:
    • Read Master Sheet
    • Read Client Sheet
    • Write to n8n Sheet
    • Mark Row as Done
  5. Click "Save" after updating each node
‚ö†Ô∏è Important

If you skip this step, the workflow will fail with authentication errors.

2.4: Add OpenAI Credentials

  1. Go to "Credentials" ‚Üí "Add Credential"
  2. Search for "OpenAI"
  3. Enter your OpenAI API Key
  4. Click "Save"
  5. Go back to the workflow
  6. Click on the "Generate with OpenAI" node
  7. Select your OpenAI credential
  8. Click "Save"

üöÄ Step 6: Activate the Workflow

Start Automation

  1. In n8n, open the "Veo3 Master" workflow
  2. Click the "Active" toggle switch (top-right)
  3. The workflow will now run automatically every 1 minute
‚úÖ What Happens Now

Every minute, the workflow will:

  1. Read the Master Sheet for client information
  2. Check each client's Sheet1 for unprocessed rows
  3. Generate AI prompts using OpenAI
  4. Write processed data to the client's n8n sheet
  5. Mark rows as "Done" in Sheet1

Monitor Workflow Execution

  • Click "Executions" in the left sidebar to see workflow runs
  • Green = Success, Red = Error
  • Click on any execution to see detailed logs

üîß Troubleshooting

Workflow Errors

Problem: "Authentication failed"

Solution:

  • Check that Google Service Account credential is properly configured
  • Verify all Google Sheets nodes have the credential selected
  • Ensure service account email has Editor access to sheets
Problem: "Sheet not found"

Solution:

  • Check sheet name is exactly "Credentials" in Master Sheet
  • Verify sheet names "Sheet1" and "n8n" in client sheets
  • Confirm URLs in Master Sheet are correct
Problem: "OpenAI API error"

Solution:

  • Check OpenAI credential is configured
  • Verify API key is valid and has credits
  • Check OpenAI API usage limits

Client Extension Issues

Problem: Client can't log in

Solution:

  • Verify their email is in the Master Sheet
  • Check spelling of email (case-sensitive)
  • Ensure Master Sheet is shared with service account
Problem: No data showing in extension

Solution:

  • Verify n8n workflow is active
  • Check client's sheet URL in Master Sheet
  • Ensure client's sheet has "n8n" tab
  • Confirm n8n has processed some rows

Common Support Questions

Q: How do I change the workflow frequency?

A: Edit the "Every 1 Minutes" node and adjust the interval.

Q: Can clients have multiple sheets?

A: No, each client should have one sheet URL in the Master Sheet.

Q: How do I remove a client?

A: Simply delete their row from the Master Sheet. Their sheet will no longer be processed.
